Твоя задача делать запросы на сторонние api по расписанию, получать фильмы и отображать их у себя. 
1)  Локализация
2) возможность покупать подписку для того чтобы смотреть новые фильмы раньше других
3)  Админка( не джанго) , кастомная которая дает возможность управлять контентом
4)  Типы пользователей:
    1. Обычный пользователь
    2. Менеджер (который управляет контентом) 

    3. Главный админ( создает и удаляет менеджеров)  
5) комментарии
6)  рейтинг
7)  получение свежей информации о фильмах через api кинотеатров 
8)  мониторить статистику посещений и просмотров конкретного фильма
9)  на основе 8 пункта каждую неделю генерировать отчет в котором в каких либо графиках отображается статистика ( популярные фильмы,  страны с которых посещали сайт, половое соотношение)
